# Batchr
Batchr is the first truly ambient operating system for restaurants. 

## Features
- Automated Ingredient Restocking
- Smart Recipe Creation + Tracking
- Optimized Production Scheduling
- Employee Management

# Running Locally
To run this project locally, run `npm run dev` from the src folder to start the vite frontend server
Then, run `cd backend` to go into the backend folder and `npm run dev` again to start up the express routes and MongoDB database
